API ENDPOINTS
═════════════════════════════════════════════════════════════════

POST /api/auth/send-otp
├─ Purpose          : Send OTP to user's email
├─ Rate Limit       : 3/min per IP, 1/min per email
├─ Response (200)   : { success, message, email (masked), expiresIn }
└─ Response (429)   : { success, message, retryAfter }

POST /api/auth/verify-otp
├─ Purpose          : Verify OTP code from email
├─ Rate Limit       : 10/min per IP
├─ Response (200)   : { success, message, email, verified }
└─ Response (400)   : { success, message, attemptsRemaining }
